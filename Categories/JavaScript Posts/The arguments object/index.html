The arguments object

What are arguments

Function arguments are real values passed to the function to obtain a result.

Arguments vs Parameters

Function parameters are the names listed in the function's definition. Function arguments are the real values passed to the function when it is called. The function arguments replace the function parameters in the function call. NOTE: parameters are in the function definition and arguments are in the function call.
